# Pac-Man-Game
Console-based Pacman game in C++ with simple ASCII graphics. Control Pacman using arrow keys, collect all dots, and avoid ghosts moving on fixed paths. Easy to compile and play on Windows.
A simple **Pacman** game implemented in **C++** for the console. Navigate Pacman through the maze, collect all the dots, and avoid ghosts moving along fixed paths. This project demonstrates **basic C++ programming concepts**, including loops, conditionals, functions, arrays, and console input/output.

## Features

- Console-based gameplay with ASCII graphics
- Player-controlled Pacman using **arrow keys**
- Ghosts move on **predetermined paths** 
- Score tracking and lives system
- Easy to compile and run on Windows
